# MinuteRepeater
A Java minute repeater.
Used traditionally on pocket watches and some wrist watches to tell the time without looking at the screen

Chimes the time.
First chimes the hour, then quarter and minute to interprete the time. 

For Example, if the time is 5:37, then there would be 5 chimes to represent the hour, 2 Chimes to reprsent that 2 15 minute quarters have passed, and then 7 chimes to present that it is 7 mins into the 3rd quarter.
